ACURA MDX 2020

1. Set the power mode to ON (do not start the engine).

2. Use the scroll wheel marked (i) on the right side of the steering wheel and select MAINTENANCE INFO.

3. Select MAINTENANCE RESET.

4. Select RESET. The displayed maintenance items disappear, and the engine oil life display returns to 100%.

5. Turn the power mode off and then start the engine to verify the indicator has been reset.
